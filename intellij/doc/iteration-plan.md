
# NextGen Point Of Sale system - iteration plan 

## Plan for phase #2 (elaboration part 1)

Use cases in decreasing priority order:

1. __Process sale (pay by card)__ - Processing sales is the most crucial use case, as there is no point of sale system without it. Within it, paying by credit card is the most common scenario. It is also technically challenging, as it requires connecting to payment authorization service. 
2. __Process sale (pay by cash)__ - Paying by cash is second most common option.
3. __Process returns__ - Important, but not as crucial as processing sales.
4. __Process sale (pay by coupon)__ - Nice-to-have feature, but not crucial.

In phase #2, we hope to create a prototype featuring the first two use cases.
